## Descripción del proyecto.
En mi proyecto lo que hicimos fue un sistema donde hay dos tipos de usuarios los cuales son admin y el usuario normal, si estás ingresado en nuestro sistema como un usuario normal el sistema solo te dará la bienvenida, si eres admin te dará opciones de administrador del sistema.

## Tecnologías utilizadas.
En mi proyecto utilizamos Python en la creación de métodos y utilizamos MySQL para la creación de la base de datos

# Instrucciones de instalación y ejecución.
## Paso 1
Guardar la carpeta con el proyecto y abrirla en visual code
## Paso 2
Verificar la contraseña del mysql en el archivo conexion.py
## Paso 3 
abrir la terminal con comand prompt y ejecutar el siguiente comando: python main.py
## Paso 4
Ejecutar Sistema
